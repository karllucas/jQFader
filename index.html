jQFader

A simple jQuery plugin to make fading out elements

jQFader v1.0

What you need

jQuery library v1.4+ : http://jquery.com/

How to use it

jQuery(".yourclass").jqfader();

.yourclass{ display:none; }

Parameters

Name Value Default
speed number 500
randomize false|true false
reverse false|true false
animate number 500
callback function void

You can find examples here : http://git.mgeoffray.fr/jQFader/
